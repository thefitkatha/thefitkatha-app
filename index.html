1. Open your FitKatha Google Sheet
2. Go to Extensions → Apps Script
3. Paste the FitKatha_Backend.gs code
4. Click Deploy → New Deployment → Web App
5. Set access to "Anyone" → Deploy
6. Copy the URL and paste above
